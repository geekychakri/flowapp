A Minimalist Productivity App

## Features

- Ambient Sounds
- Pomdoro Timer
- Todo List
- Timeboxing
- Notes
- LoFi Radio

## Run Locally

Clone the project

```bash
  git clone https://github.com/GeekyChakri/flowapp.git
```

Go to the project directory

```bash
  cd flowapp
```

Install dependencies

```bash
  npm install
```

Create an .env file in root and add your variables

```
  AUTH0_SECRET=
  AUTH0_BASE_URL=
  AUTH0_ISSUER_BASE_URL=
  AUTH0_CLIENT_ID=
  AUTH0_CLIENT_SECRET=
  DB=
  DB_KEY=
```

Start the app

```bash
  npm run dev
```
